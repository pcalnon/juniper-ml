
# juniper-canopy left menu

## Plan development

develop a detailed plan to perform the following:

modify the Training Parameters section of the left menu to provide more precise and more granular and more varied settings

### Enhancement Requirements

make changes including the following:

- change the Training Parameters section Heading to Meta Parameters
- create two subsections:
  - Neural Network subsection
  - Candidate Nodes subsection
- both subsections should be able to be collapsed/expanded
- both subsections should be wired up to use a shared Apply Parameters button

#### Neural Network

make changes including the following:

- Neural Network subsection should include the following elements:
  - maximum iterations (network growth and output training)
    - numeric field, int, default: 1000
  - maximum total epochs
    - numeric field, int, default: 1000000
  - learning rate
    - numeric field, float, default: 0.01
  - maximum hidden units
    - numeric field, int, default: 1000
  - multi-node layers
    - checkbox, default: unchecked
  - network growth triggers
    - preset epochs
      - radio button, linked to convergence detection, default: unselected
      - number of epochs
        - numeric field, int, default: 50
    - convergence detection
      - radio button, linked to preset epochs, default: selected
      - convergence threshold
        - numeric field, float, default: 0.001
  - spiral dataset
    - spiral
      - rotations
        - numeric field, float, default: 1.5
      - number
        - numeric field, int, default: 2
    - dataset
      - elements
        - numeric field, int, default: 1000
      - noise
        - numeric field, float, default: 0.25

#### Candidate Nodes

make changes including the following:

- Candidate Nodes subsection should include:
  - Candidate Pool size
    - numeric field, int, default: 100
  - Correlation threshold
    - numeric field, float, default: 0.001
  - Selected Candidates
    - numeric field, int, default: 1
  - Pool Training Complete
    - preset epochs
      - radio button, linked to convergence detection, default: selected
      - Training iterations
        - numeric field, int, default: 500
    - convergence detection
      - radio button, linked to preset epochs, default: unselected
      - convergence threshold
        - numeric field, float, default: 0.0001
  - Multi Candidate Selection
    - checkbox, linked to multi-node layers in the neural network subsection--checking box also checks multi-node layers, default: unchecked
      - Add top tier candidates
        - radio button, linked to add random candidates, default: unselected
        - number of top candidates
          - numeric field, int, default: 1
      - Add random candidates
        - radio button, linked to add top tier candidates, default: unselected
        - number of random candidates
          - numeric field, int, default: 1

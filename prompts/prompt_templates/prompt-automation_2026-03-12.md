
# Juniper Prompts

Perform an investigation into all Juniper project prompts, including:

- Juniper/juniper-ml/prompts/
- Juniper/prompts/ and its subdirectories

## Primary Objective

### Plan Development

Develop a plan to perform the following:

- investigate Juniper project prompts
- perform detailed and rigorous analysis of the prompts
- evaluate various mechanisms and approaches for automating various aspects of Juniper prompts:
  - prompt creation
  - prompt modification
  - prompt use
- research possible approaches and suggested best practices including but not limited to:
  - constructing prompt type templates
  - database use for specific elements to populate prompt template details
  - utility scripts to perform environment discovery
- suggest a variety of approaches to these automation goals
- evaluate the strengths, weaknesses, risks, and guardrails of each approach
- include discussions of possible implementation details

### Prompt Analysis

Conduct a detailed analysis of all project prompt files.
Focus particular attention on the following:

- Identifying Common elements across various sets of prompts
- Identify Common approaches that are used frequently
- Identify Common goals for various groups of prompts
- Identify Elements of redundancy within prompts and among similar sets of prompts
- Identify Common elements of content, format, and layout
- Identify natural boundaries between various types of prompts

### Automation Analysis

Expand the analysis to research and consider various approaches to prompt automation

- research prompt automation topics
- consider strengths, weaknesses, risks, and guardrails
- evaluate implementation approaches
- consider potential support infrastructure
- consider maintenance of documentation
- evaluate methods for assuring repeatability
- prioritize transparency, maintainability, and flexibility

### Conclusions and Recommendations

Document all aspects of this analysis
Rate and recommend various potential solutions
Write the plan into a markdown file in the Juniper/juniper-ml/notes/ directory
utilize sub-agents to validate all aspects of this plan

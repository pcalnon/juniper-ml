### Consolidate, Synthesize, and Validate loop

Starting with an initial number of 10 sub-agents specialized for the task, perform the following:

1. Each sub-agent should evaluate all of the currently active proposals.
2. Each sub-agent should then generate a new, more comprehensive, more detailed, and more accurate proposed development plan.
3. Each sub-agent's proposed development plan should be written to a uniquely named markdown file.
4. Each sub-agent should validate all aspects of their proposed development plan, making corrections and/or additions as applicable.
5. These proposed development plans, created by each of this current group of sub-agents, will become the new set of active proposals.
6. The new set of active proposals will be used by a new group of sub-agents consisting of 1 fewer sub-agents than the previous group.
7. The new group of sub-agents will repeat this process beginning with Step 1.
8. This process should continue until the current group of sub-agents and the number of newly created proposed development plans is 1.

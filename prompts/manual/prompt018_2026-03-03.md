# Juniper Cascor Tests issues

## Primary Objective

Investigate test issues with the juniper-cascor tests suite, determine root causes, address and resolve the issues.

## Directives

Addressing these juniper-cascor test issues should NOT involve deleting tests, disabling tests, or removing or commenting out specific checks.

## Deliverables

1. Develop a plan to deal with all juniper-cascor test issues.
    - Plan should cover steps to identify, investigate, verify, analyze, fix, and validate all test suite issues.
    - Plan should include, but not be limited to, the 4 issue described below
2. Write the plan into a markdown file in the notes/ directory.
3. Execute the plan.
4. validate that all of the fixes succeeded, that all tests pass, and that there are no remaining test issues.

# Enhance and Document Juniper Startup

## Primary Objective

develop a plan to enhance and document the full project launch process for the Juniper project

## Startup Modes

**launch modes:**

run on host with manual launch
run on host as services
run in containers with docker compose
run in dedicated demo mode
run as backend only
run with canopy frontend connecting to already running backend

## Deliverables

### Scripts

**Scripts Tasks:**

1. Create a full set of convenience scripts for launching the entire juniper project (or single application) in all avaiable startup modes.
2. Scripts should be structured and configured in a manner that follows the existing conventions as the utility scripts
    - same logging tuilized
    - same config files used, both common and script specific

### Documentation

**Documentation Tasks:**

1. Create full startup documentation and add content or references as appropriate to all existing applications' documentation.
2. Update the developer cheatsheet to reflect changes made in this prompt.

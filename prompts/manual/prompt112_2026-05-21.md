# PyPI meta package expansion

The primary goal of this thread is to extend the functionality of the juniper-ml meta-package.

## Desired Functionality

### Juniper Servers

Currently, the juniper-ml meta package, that is deployed to pypi, does not allow installation of the juniper project servers:

- juniper-canopy
- juniper-cascor
- juniper-data

Ideally, running the following command should succeed and install all three of the servers listed above:

```bash
(JuniperCanopy1) pcalnon@yamaguchi:~/Development/python/Juniper/juniper-canopy$ pip install juniper-ml[servers]
Requirement already satisfied: juniper-ml[servers] in /opt/miniforge3/envs/JuniperCanopy1/lib/python3.13/site-packages (0.4.1)
WARNING: juniper-ml 0.4.1 does not provide the extra 'servers'
```

### Juniper Tools

Additionally, juniper-ml should be able to install the tool packages listed below:

- juniper-ci-tools
- juniper-doc-tools
- juniper-observability

Running the following command should successfully install the three project tool packages listed above:

```bash
(JuniperCanopy1) pcalnon@yamaguchi:~/Development/python/Juniper/juniper-canopy$ pip install juniper-ml[tools]
Requirement already satisfied: juniper-ml[tools] in /opt/miniforge3/envs/JuniperCanopy1/lib/python3.13/site-packages (0.4.1)
WARNING: juniper-ml 0.4.1 does not provide the extra 'tools'
```

### All Juniper Packages

Finally, using pip install juniper-ml[all] should install all of the clients, workers, servers, and tools packages

- juniper-canopy
- juniper-cascor
- juniper-data
- juniper-data-client
- juniper-cascor-client
- juniper-cascor-worker
- juniper-ci-tools
- juniper-doc-tools
- juniper-observability

As above, installing all of these packages should be possible using the following command:

```bash
(JuniperCanopy1) pcalnon@yamaguchi:~/Development/python/Juniper/juniper-canopy$ pip install juniper-ml[tools]
```

## Existing Functionality

Currently, the juniper-ml meta-package will only allow installs of the following 3 package groups:

- clients
  - juniper-data-client
  - juniper-cascor-client
- worker
  - juniper-cascor-worker
- all
  - juniper-data-client
  - juniper-cascor-client
  - juniper-cascor-worker

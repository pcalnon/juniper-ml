# Evaluate design of new Microservices Architecture

## specific areas of analysis

### Coordinated Application startup

Develop a plan for Juniper project launch/startup
consider various approaches to launching the Juniper project's applications, services, and clients
evaluate each approach with respect to:

- how cleanly would each of the options for startup of the various microservices be orchestrated.
- how effectively would these approaches scale as new services and clients are added
- evaluate how well each option conforms to best practices
- consider security risks and performance issues related to each option

### Modes of Operation

perform an in-depth analysis of the purpose and usefullness of the current two distinct scenarios:

- normal training of a juniper-cascor neural network
- demo mode to showcase the functionality and features of the juniper-canopy frontend for juniper-cascor

Suggest multiple options for what these operating modes mean and how they could be implemented in a microservices architecture.

- evaluate how well each option conforms to best practices
- consider security risks and performance issues related to each option
- evaluate how well each option scales
- consider how transparent and maintainable each option is

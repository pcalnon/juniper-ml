# Refactor Cascor Network and Candidates

## Refactor Overview

Create a detailed plan to perform a very specific refactor of the juniper-cascor application's CascadeCorrelationNetwork class and CandidateUnit class

## Refactor Description

**There are two aspects to the refactor:**

1. extract all non-Cascade Correlation Neural Network Model specific structures and functionality into its own, network management class.
    - this should include the following:
      - remote worker management
      - parallelism
      - queue management
      - candidate pool training
      - candidate pool management
      - functionality to calculate correlation values
      - interaction with data clients
      - interaction with dataset loading
      - dataset padding or adjustment

2. extract all Cascade Correlation Neural Network Model specific structure and functionality into its own class.
    - this should include the following:
      - output layer training
      - forward processing
      - integrating node (or nodes) into network
      - modifying network topology for live data swaps
      - inference
      - the network model itself
      - error calculation
      - accuracy calculation

## Problem Description

The Dataset View tab of the Juniper Canopy Dashboard is not displaying the current dataset.

Opening the Dataset View tab results in the following attribute error:

```bash
AttributeError: 'JuniperCascorClient' object has no attribute 'get_dataset_data'. Did you mean: 'get_dataset'?
```

The JuniperCascorClient class is defined in the juniper-cascor-client application's juniper_cascor_client directory and client.py file.
On line 219 of the client.py file, the get_dataset_data() method is defined:

```python
    def get_dataset_data(self) -> Dict[str, Any]:
        """Get dataset arrays (train_x, train_y, optionally val_x, val_y) for visualization."""
        return self._get("/dataset/data")

```

# juniper-cascor pre-commit checks

## Overview

the Juniper/juniper-cascor application is currently raising errors during testing.

## Directives

perform a detailed analysis of the juniper-cascor codebase and develop a plan to do the following:

- investigate these issues
- analyze the codebase and error messages
- identify root causes
- identify any other potential issues with the codebase
- develop fixes to address all root causes and identified issues
- add tests as needed to validate all fixes
- ensure all tests succeed
  - do NOT disable, delete, or comment out any testing functionality or files

use sub-agents to validate all aspects of this plan
write the plan into a markdown document in the Juniper/juniper-cascor/notes/ directory
once the plan has been validated and written to a file, implement the plan
